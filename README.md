# ACADEFICA: Retenção Inteligente (Landing Page)

Esta é a Landing Page para a startup ACADEFICA, que utiliza Machine Learning (IA) para prever e reduzir a evasão de alunos em academias.

O projeto foi desenvolvido em React com TypeScript, utilizando o Vite como *bundler* e o Ant Design para os componentes de UI, seguindo um design moderno com fundo escuro e cor principal amarela.

---

## Tecnologias e Design

| Categoria | Detalhe |
| :--- | :--- |
| **Framework** | React (com Vite) |
| **Linguagem** | TypeScript |
| **Biblioteca UI** | Ant Design (antd) |
| **Tipografia** | Poppins (Google Fonts) |
| **Paleta** | Tema Escuro (`#0D0D0D`) e Amarelo Vibrante (`#FFD700`) |

## Configuração e Instalação

Siga os passos abaixo para configurar o projeto na sua máquina local.

### Pré-requisitos

Certifique-se de ter instalado o **Node.js** (versão 18 ou superior) e o **npm** (Node Package Manager) ou **Yarn**.

### 1. Clonar o Repositório

Clone o repositório para sua máquina local 

### 2. Instalar Dependências

Instale todas as bibliotecas necessárias listadas no package.json:

```npm install```

## Como rodar o projeto

Após instalar as dependências, utilize o script de desenvolvimento do Vite.

```npm run dev```

O servidor de desenvolvimento será iniciado.
Acesse a Landing Page em: http://localhost:5173/ (ou a porta exibida no terminal).
